### 🚀 如何测试聊天功能

#### 步骤 1: 打开两个客户端
已经在浏览器中打开了两个聊天客户端窗口！

#### 步骤 2: 注册/登录用户

**客户端 1 (Alice):**
- 用户名: `alice`
- 密码: `Alice123`
- 点击 "Register" 注册

**客户端 2 (Bob):**
- 用户名: `bob`
- 密码: `Bob123`
- 点击 "Register" 注册

#### 步骤 3: 开始聊天

1. **Alice** 在消息框输入: "Hello Bob!"
2. **Bob** 在消息框输入: "Hi Alice!"
3. 消息会实时显示在两个客户端中！

### 🔧 技术架构

#### 消息流程
```
客户端1 (Alice) 
    ↓
    HTTP POST (发送消息到服务器)
    ↓
    保存到 SQLite 数据库
    ↓
    MQTT Publish (发布到主题)
    ↓
    MQTT Broker (转发消息)
    ↓
    订阅者收到消息
    ↓
客户端2 (Bob)
```

#### MQTT Topics
- `chat/group/{groupId}/message` - 群组消息
- `chat/private/{fromId}/{toId}` - 私聊消息
